# 🎬 MOBI-HUBB - Movie Download Site

## 📌 About
MOBI-HUBB is a movie download website where users can find and download HD movies, web series, and TV shows.

## 🚀 Features
- 735 Pages with Pagination
- Search Functionality
- Movie Details Page
- Adsterra Smart Links
- Bookmark Popup
- Mobile Responsive
- Dark Theme

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Cloudflare Pages
- GitHub

## 📦 Deployment
This site is deployed on Cloudflare Pages with automatic sync from GitHub.

### Auto-Deploy Workflow
1. Push changes to `main` branch
2. Cloudflare Pages automatically builds and deploys
3. Live site updates within 1-2 minutes
